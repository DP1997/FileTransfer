1. Aufgaben Woche 5

In diesen Aufgaben wollen wir uns mit den Effekten der dynamischen Speicherverwaltung beschäftigen.

1.1. Aufgabe 1

Betrachten Sie die folgenden beiden Programm os01.cc und os02.cc:

Quelltext 1. Zeilen aus der Datei os01.cc [Snippet: p1] (einige Zeilen nicht dargestellt)
// Festlegung der POSIX-Version und notwendige #include-Anweisungen
const size_t innerSize = 4096;
const size_t outerSize = 4096 * 100;
unsigned char array[outerSize][innerSize] = {};
int main() {
#ifdef PAGESIZE
    const long pageSize = PAGESIZE;
#else
    const long pageSize = sysconf(_SC_PAGE_SIZE);
#endif
    printf("pagesize: %ld\n", pageSize);
    for(size_t i = 0; i < outerSize; i += 1) {
        for(size_t k = 0; k < innerSize; k += 1) {
            array[i][k] = static_cast<unsigned char>((i * k) % 256);
        }
    }
    exit(EXIT_SUCCESS);
}
Quelltext 2. Zeilen aus der Datei os02.cc [Snippet: p2] (einige Zeilen nicht dargestellt)

Analysieren Sie den obigen Code und beantworten Sie folgende Fragen:

  • Wie groß ist das Array unsigned char arry[outerSize][innerSize]; in Bytes im virtuellen Adressraum.?

  • Gibt es einen Unterschied der Werte in den Arrayelementen, nachdem die verschachtelte Schleifenstruktur abgelaufen ist?

  • Welches der beiden Programme wird schneller / langsamer laufen?
